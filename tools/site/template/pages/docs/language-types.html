ZSCRIPT is a modern, dynamically typed scripting language designed for flexibility, ease of use, and powerful runtime features. It offers a hybrid type system, combining the best of both weak and strong typing, giving developers the freedom to choose between dynamic flexibility and static type safety when needed.

The zscript language is an hybrid between weakly and strongly typed. Basic types are `null`, `bool`, `integer`, `float`, `string`, `table`, `array`, `function`, `class`, `instance` and `userdata`. A variable declared with {{zscript-intext-code "var"}} is weakly typed and can be assigned or reassigned any type.

A variable declared with {{zscript-intext-code "var"}} is weakly typed and can be assigned or reassigned any type. {{#code-block label="load_file"}}var a = null; assert(typeof(a) == "null"); a = 1; assert(typeof(a) == "integer"); a = "abc"; assert(typeof(a) == "string");{{/code-block}}

A variable declared with a type, is strongly typed and can only be assigned the same type. {{#code-block label="load_file"}}int a = 51; assert(typeof(a) == "integer"); // ERROR - Trying to assign a string value to an integer variable. a = "abc";{{/code-block}} A variable declared with the {{zscript-intext-code "auto"}} keyword is also strongly typed but the assigned type is automatically detected. {{#code-block label="load_file"}}auto a = 31; assert(typeof(a) == "integer"); // ERROR - Trying to assign a string value to an integer variable. a = "abc";{{/code-block}} It is also possible to restrict a variable assignable types. {{#code-block label="load_file"}}var a = 31; assert(typeof(a) == "integer"); a = 31.1; assert(typeof(a) == "float"); // ERROR - Trying to assign a string value to a var variable. a = "abc";{{/code-block}}
