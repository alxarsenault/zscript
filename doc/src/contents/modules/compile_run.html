Compiling a file will give us a closure that will execute the content of the file when called.

We assign the dummy.zs file closure to the dummy1_closure variable. The closure contains a some byte code, generated by the compiler, that will be executed everytime it is called. Calling dummy1_closure() executes and returns the value i.e. a table. 

const dummy1_closure = zs.compile_file("dummy1");
  const dummy1_content = dummy1_closure();
  dummy1_content.do_something(2, 3);
The zs.compile_file function will recompile and create a new closure everthing time.
