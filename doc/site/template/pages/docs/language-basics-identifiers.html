You use variables as symbolic names for values in your application. The names of variables, called identifiers, conform to certain rules.

Identifiers start with an alphabetic character or the symbol {{zscript-intext-code "_"}} followed by any number of alphabetic characters, {{zscript-intext-code "_"}} or digits [0-9]. {{zname}} is a case sensitive language meaning that the lowercase and uppercase representation of the same alphabetic character are considered different characters. For instance, {{zscript-intext-code "foo"}}, {{zscript-intext-code "Foo"}} and {{zscript-intext-code "FOo"}} are treated as 3 distinct identifiers.

You can use most Unicode letters such as å and ü in identifiers. (For more details, see the lexical grammar reference.)

Some examples of legal names are {{zscript-intext-code "Number_hits"}}, {{zscript-intext-code "temp99"}} and {{zscript-intext-code "_name"}}.
